# Svelte + Vite + TailwindCss

## Start Project in Dev
1. git clone https://github.com/drexqq/toy-project-1.git
2. cd app
3. npm install
4. npm run dev
5. localhost:5374

## Environment
1. npm (8.15.0)
2. node (16.17.0)

## Library
1. tailwindcss
2. postcss
3. autoprefixer
